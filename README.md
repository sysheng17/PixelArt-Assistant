# 🎨 PixTown Assistant | 心動小鎮畫作助手

![Language](https://img.shields.io/badge/Language-HTML%2FJS%2FCSS-orange)

PixTown Assistant 是一個輕量、強大的前端圖片處理工具，旨在幫助玩家將現實照片或插畫轉化為高品質的像素風格參考圖。特別針對《心動小鎮》遊戲內的 125 色畫板進行了色彩匹配優化。

https://sysheng17.github.io/PixelArt-Assistant/

## ✨ 功能特色

- 🛠️ **精準裁切**：內建常見比例（16:9, 4:3, 1:1...），完美適配各種畫布規格。
- 📏 **動態像素化**：自由輸入長寬，支援高達 150px 以上的自定義解析度。
- 🌈 **專業色彩處理**：
  - **心動小鎮 125 色盤**：精準匹配遊戲內顏料。
  - **Floyd-Steinberg 抖動演算法**：讓色彩過渡更自然，層次感更豐富。
  - **標準 256 色 & 復古 16 色**：支援多樣化創作風格。
- 📏 **網格導出**：支援導出帶有參考線的圖片，方便對照繪製。
- 🌍 **多國語言**：一鍵切換 繁體中文 / English。

## 🚀 快速開始

1. **上傳圖片**：選擇你想要轉化的圖片。
2. **剪裁區域**：調整至合適的構圖。
3. **設定尺寸**：選擇預設尺寸或手動輸入像素寬高。
4. **色彩處理**：選擇「小鎮125色」或其他色彩模式，開啟「抖動」獲得另類效果。
5. **下載成果**：選擇純像素圖或帶有參考線的網格圖。

## 🛠️ 技術棧

- **Core**: Vanilla JavaScript (ES6+)
- **Image Cropping**: [Cropper.js](https://github.com/fengyuanchen/cropperjs)
- **Styling**: CSS3 (Modern UI Style)
- **Algorithms**: Floyd-Steinberg Dithering, Euclidean distance color matching.
